Installation

From download to first run in under a minute.

Download the DMG

Pick the build for your Mac from the buttons above — Apple Silicon or Intel.

Drag to Applications

Open the DMG and drag Unifize Test Runner into your Applications folder.

Bypass Gatekeeper (first launch only)

The app is unsigned, so macOS may block it. Either right-click the app → Open → Open, or run this once in Terminal:

cd ..
xattr -dr com.apple.quarantine "Unifize Test Runner.app"

Pick your repo

On first launch a folder picker appears — select your playwright-tests repo. The path is saved to ~/.unifize-test-runner.conf and reused on every future launch.

Keychain prompt: On first run macOS may ask for keychain access for git credentials — click Always Allow so it won't prompt again.

How to use it

Click Run Tests and step through the 6-step wizard.

StepWhat you choose
1 — EnvironmentProduction (app.unifize.com) or Staging
2 — BranchSwitch branch + auto-sync from upstream
3 — Module / FolderUniversal search — finds folders and spec files
4 — Spec FilesMulti-select specs (Cmd+click), or double-click to select & continue
5 — Browser ModeHeadless / Headed / Debug (Playwright inspector)
6 — Workers & RetriesParallel workers (1 / 2 / 4) and retry count (0 / 1 / 2)
Tip: Double-click any option at any step to select it and jump to the next step automatically. Hit ▶ Run Tests to stream live output, then Open Report → for the full HTML report.

Building from source

Prefer to build it yourself? Clone the repo and run:

git clone https://github.com/prasadhelaskar-unifize/playwright-runner.git
cd playwright-runner
npm install        # install dependencies
npm start          # run in development mode
npm run build      # build standalone macOS DMG → dist/
